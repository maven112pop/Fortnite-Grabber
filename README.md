# FN Grabber 🎮

**FN Grabber** is a sleek Python script that lets you grab the **Account ID** of any Fortnite player by entering their username and platform (epic, xbl, or psn).

---

## 🔧 Requirements

- Python 3.x
- Internet connection

---

## 📌 Dependencies

- requests
- beutifulsoup4

---

Install Dependencies by:

- Open CMD
- cd C:\Users\[YOURNAME]\Downloads\Fortnite Grabber
- pip install -r requirements.txt

---

## 🚀 How to Use

1. Run Grabber.exe
2. Enter the Fortnite username and platform when prompted.
3. The script will fetch and display the account ID.

---

## 📁 Files

- `fn_grabber.exe` – main script
- `requirements.txt` – required Python packages
- `README.md` or `README.txt` – this file
